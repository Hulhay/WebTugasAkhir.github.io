Pendahuluan

Pengukuran parameter fisis batuan berpori memiliki peranan penting dalam karakterisasi reservoir. Pengukuran parameter fisis secara konvensional di laboratorium masih memiliki kekurangan karena membutuhkan waktu yang lama dan berpotensi merusak batuan berpori itu sendiri.

Batuan digital dapat menjadi media alternatif dalam pengukuran parameter fisis batuan berpori. Beberapa metode yang dapat digunakan untuk mengukur parameter fisis batuan digital diantaranya adalah simulasi numerik dan Machine Learning.

Metode Penelitian

Data
Collection

Mengumpulkan data batuan digital dari Digital Rocks Portal.

Data
Cleansing

Menyiapkan data agar dapat digunakan sebagai data latih dan evaluasi.

Hyperparameter
Tuning

Menentukan Hyperparameter ML terbaik.

Model
Training

Melatih model Machine Learning dengan menggunakan data latih.

Model
Evaluation

Menguji model Machine Learning menggunakan data evaluasi.
